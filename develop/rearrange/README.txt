== 使い方 ==

1. 管理画面の「外観」→「テーマ」→「新規追加」→「テーマのアップロード」の順に進み、
親テーマと子テーマを両方インストールしてください。
2. インストールが完了したら、子テーマを有効化してください。
3. 左側のメニューに「Godios」という項目ありますのでそちらから設定を行ってください。
4. デザインは「外観」→「カスタマイズ」から行ってください。
5. 画像を変更したい場合は、子テーマのimagesディレクトリのファイルを上書きしてください。
